# erc20_token_test
Test for creating an ERC20 cryptocurrency in Ethereum

## Requisites
* [NodeJS](https://nodejs.org/)

## Installation
```
npm install
```

## Usage
Test:
```
npx ganache-cli
npx truffle test
```
Deploy:
```
npx truffle migrate --reset
```
